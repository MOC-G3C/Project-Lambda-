# DATA VALIDATION CHECKLIST (QC-02)
**Objective:** To filter and verify information before official integration into the Axis [cite: 2026-01-21].

## 1. Source Reliability
* [ ] **Identity:** Is the origin of the data clearly identified and trustworthy?
* [ ] **Traceability:** Can we find the primary source if needed?
* [ ] **Bias Check:** Is the information objective or influenced by a specific agenda?

## 2. Accuracy & Logic
* [ ] **Verification:** Has the data been cross-referenced with at least one other independent source?
* [ ] **Consistency:** Does this new data conflict with the existing "Physical Anchor" (Pillar 1)?
* [ ] **Simplicity:** Is the data expressed clearly, or is it hidden behind complex jargon? [cite: 2026-01-21]

## 3. Practical Utility
* [ ] **Relevance:** Does this information directly help achieve the current project objective?
* [ ] **Actionability:** Can we actually use this data to make a decision or a calculation?
